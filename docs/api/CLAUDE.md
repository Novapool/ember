# API Documentation

Integration guides and endpoint documentation for Bonfire services.

## Available Documentation

- **FIREBASE.md** - Firebase Realtime Database setup and integration
  - Read when: Setting up Firebase, deploying to production, local development with emulator
  - Covers: Firebase project creation, service account setup, emulator usage, production deployment (Heroku, Railway, Render, GCP), environment configuration, troubleshooting

- **ADMIN_API.md** - Admin REST endpoints for server management
  - Read when: Setting up admin endpoints, managing servers in production, implementing monitoring
  - Covers: Health checks, server statistics, force-end rooms, kick players, API authentication
